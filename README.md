# 📝Task Tracker

Ein kleines Lernprojekt mit HTML, CSS und JavaScript.

## 🎯 Ziel des Projekts

Dieses Projekt wurde erstellt, um die Grundlagen von: 

- HTML (Struktur)
- CSS (Layout & Styling)
- JavaScript (DOM Manipulation & Events)
- Git & GitHub (Versionierung)

praktisch zu üben.

---

## 🚀 Funktionen

- ✅ Aufgaben hinzufügen
-  🗑 Aufgaben löschen
-  ✔ Aufgaben als erledigt markieren
- 💾 Speicherung im Browser (LocalStorage)
- ⌨ Unterstützung für Enter-Taste

---

## 🧠 Was ich gelernt habe

- Dynamisches Erstellen von HTML-Elementen mit JavaScript
- Arbeiten mit `addEventListener`
- Nutzung von `classList.toggle()`
- Arbeiten mit Arrays als State
- Nutzung vin `LocalStorage`
- JSON.Stringify() und JSON.parse()
- Strukturierte Git-Commits
- Projektorganisation

---

## 🛠 Technologien

- HTTML5
